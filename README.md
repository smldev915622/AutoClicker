Auto Clicker is a Chrome extension for recording viewport click positions and replaying them with custom per-point delays on the active tab only.

HOW IT WORKS
1. Open the extension on your tab
2. Click Record Position and click spots on the page
3. Set delays for each point
4. Click Start or use your hotkey
5. Stop anytime with Stop, Stop All, or Emergency Stop

FEATURES
• Visual recording — no manual X/Y typing
• 40+ click points with enable/disable, reorder, and delete
• On-page markers (full, dots only, or hidden)
• Saved coordinate presets
• Click modes: DOM, mouse sequence, and Canvas/WebGL
• Configurable Start, Stop, Toggle, and Emergency Stop hotkeys
• Loop forever, for N loops, or for a set duration
• Stops on tab switch, reload, or manual stop

PRIVACY
All settings, click points, hotkeys, and presets are stored locally on your device using chrome.storage.local. This extension does not collect, transmit, or sell user data.

PERMISSIONS
• storage — save settings locally
• activeTab — work only on the tab you use the extension on
• scripting — inject automation on that tab when you open the popup or start recording
• tabs — stop automation when you switch tabs or reload

DISCLAIMER
For personal automation on pages you control or are permitted to automate. Users must follow the terms of service of any website or game they use.
